Anthropogenic effects on chungungos abundance

Here we will explore some anthropogenic variables as potential drivers of abundance change on this species

Lets do some nice ggplots to examine our data before fitting fancy models:

There is a trend but the error is HUGE!

Well, it seems that the new index is significant

Following Ben Bolkers advice, we can avoid singular fit by removing those random effects with a very litle contribution (in this case, month). Lets try fitting a new model using only site as random effect:

A Bayesian approach
